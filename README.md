# Zendenta ERP

Clinical Dental ERP with secure persistence.

## Run Local
1. `npm install`
2. `npm run dev`

## Vercel Deployment
This project is optimized for Vercel:
- **Framework**: Vite
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Routing**: SPA routing configured in `vercel.json`
- **Environment Variables**: Ensure `GEMINI_API_KEY` is set in Vercel project settings.
